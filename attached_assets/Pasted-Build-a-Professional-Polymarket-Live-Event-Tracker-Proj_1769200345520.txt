Build a Professional Polymarket Live Event Tracker
Project Overview
Build a modern, professional web application that tracks Polymarket betting positions alongside real-world live data (crypto prices, weather, sports scores, stock prices). This is a SaaS platform for Polymarket traders to monitor their bets with real-time event tracking.
Tech Stack

Frontend: React with TypeScript
Styling: Tailwind CSS + shadcn/ui components
APIs:

CoinGecko API (crypto prices - free)
Open-Meteo API (weather - free, no key)
Yahoo Finance or Finnhub (stocks - free)
API-SPORTS (sports scores - free tier)


State Management: React hooks (useState, useEffect)
Data Fetching: Native fetch API with error handling
